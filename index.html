FCV Sensitivity Project Screener

A tool for screening WBG projects for recommendations to strengthen FCV sensitivity and responsiveness

This tool uses an LLM to screen draft WBG project documents for FCV dynamics and generate recommendations on how to strengthen FCV sensitivity and responsiveness. Upload your project documents and follow the step-by-step process to produce a final recommendation note. At each stage you can review the AI's thinking and make refinements before proceeding.

Output is generated by an LLM and should be approached with care. It is not a replacement for expert analysis and should be used as a supplementary input only.
