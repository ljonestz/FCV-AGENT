3-Stage Pipeline
Stage 1
Context & Extraction
Extracts FCV-relevant content from the project document (Part A) and contextualises with uploaded contextual docs + automated web research (Part B). Classifies doc type, instrument type, and temporal context.
Knowledge injected:
FCV_GUIDE PLAYBOOK_DIAGNOSTICS FCV_REFRESH FCS_LIST WB_INSTRUMENT_GUIDE FCV_GLOSSARY WB_PROCESS_GUIDE
User documents:
Project doc (required) Contextual docs incl. CPF Web research (auto)
↓
Stage 2
FCV Assessment
Applies 12 OST recommendations + 25 key questions + Peace & Inclusion Lens dimensions + DRR framing to produce thematic narrative, DNH traffic-light, S/R ratings, and Under the Hood analysis. Internal engine — TTL sees themes, not framework.
Knowledge injected:
FCV_OPERATIONAL_MANUAL FCV_REFRESH FCV_GUIDE FCS_LIST INSTRUMENT_CALIBRATION
↓
Stage 3
Recommendations Note
Generates stage-aware narrative memo + structured priority cards with actions, suggested language, and CPF alignment. Lifecycle-aware: PCN/PAD get preparation framing; AF/ISR get implementation framing.
Knowledge injected:
PLAYBOOK (stage-matched) FCV_REFRESH FCV_GUIDE CPF_INTEGRATION_GUIDE
↓
Follow-on
Go Deeper · Follow-on Queries
Per-priority deep dives (Evidence Trail from Stage 2 data; FCV Playbook via LLM call) and post-analysis tasks (peer review notes, expanded recommendations).
Key Outputs
Part A / Part B Split
Stage 1. Document-only extraction vs. contextualised analysis with tiered citations
Thematic Narrative
Stage 2. 3-5 dynamic themes + DNH traffic-light + S/R synthesis
S/R Ratings
Stage 2. Sensitivity (6-tier, OST %) + Responsiveness (6-tier, Refresh shifts)
Under the Hood
Stage 2. 12-rec table, 9-principle DNH, 25 questions, evidence trail
Priority Cards (JSON)
Stage 3. title, dimension, tag, shift, risk, gap, why, actions[], who, when, resources, pad_sections
CPF Alignment
Stage 3. Per-priority link to CPF outcome (when CPF uploaded). Renders as conditional zone in priority cards
Horizon Considerations
Stage 3. Beyond-scope FCV observations for team awareness
DOCX Report
Download. Full narrative + priority cards + CPF alignment + methodology note citing all sources
